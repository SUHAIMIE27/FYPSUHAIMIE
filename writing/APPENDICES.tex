\section{Appendix A: Dataset Description}

This study uses benchmark Dissolved Gas Analysis (DGA) datasets collected from transformer insulation systems. The dataset consists of dissolved gas concentration measurements used for fault classification.

The dataset includes key gases such as hydrogen (H$_2$), methane (CH$_4$), ethane (C$_2$H$_6$), ethylene (C$_2$H$_4$), acetylene (C$_2$H$_2$), carbon monoxide (CO), and carbon dioxide (CO$_2$). These gases are widely used for transformer condition monitoring and fault diagnosis.

The dataset is preprocessed before training, including missing value handling, normalization, and class balancing to ensure consistency and improve model performance.

\section{Appendix B: Feature List}

The following feature groups are used in this study:

\begin{itemize}
\item Raw gas concentrations (H$_2$, CH$_4$, C$_2$H$_6$, C$_2$H$_4$, C$_2$H$_2$, CO, CO$_2$)
\item Gas ratio features (e.g., CH$_4$/H$_2$, C$_2$H$_2$/C$_2$H$_4$)
\item Statistical features (mean, variance, skewness, kurtosis)
\item Density-based features
\item Transformation-based features
\end{itemize}

\section{Appendix C: Model Hyperparameters}

The machine learning models used in this study include:

\begin{itemize}
\item Support Vector Machine (SVM)
\item Decision Tree (DT)
\item Random Forest (RF)
\item k-Nearest Neighbors (k-NN)
\item Logistic Regression (LR)
\item Gradient Boosting Machine (GBM)
\item XGBoost
\end{itemize}

Hyperparameters are optimized using stratified cross-validation.

\section{Appendix D: Evaluation Metrics}

The following evaluation metrics are used:

\begin{itemize}
\item Accuracy
\item Precision
\item Recall
\item F1-score
\item Macro-F1 score
\item ROC-AUC
\end{itemize}

These metrics are selected to handle imbalanced multi-class classification problems.
