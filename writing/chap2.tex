\chapter{LITERATURE REVIEW}
This chapter reviews existing studies related to machine learning-based transformer fault classification using Dissolved Gas Analysis (DGA). The purpose of this chapter is to provide a comprehensive understanding of current methods, techniques, and challenges in DGA-based transformer diagnostics. By analyzing previous research, this chapter establishes the theoretical and technical foundation for the proposed study.

The first section discusses the application of machine learning techniques in transformer fault diagnosis, including classification models, semi-supervised learning, and data-driven approaches. It highlights how these methods are used to model the relationship between gas concentration patterns and fault types \parencite{Ren2025, Kumar2024, Nguyen2025}. The second section focuses on key components of ML-based diagnostic systems, including feature representation, dataset characteristics, and data preprocessing techniques such as data augmentation and feature selection \parencite{Du2024, Lin2026, Peng2024}. In addition, challenges such as class imbalance, limited labeled data, and model generalization are discussed based on findings from previous studies \parencite{Chen2023, Zhou2023, Wang2023}.

Furthermore, this chapter examines the role of explainable artificial intelligence (XAI) in improving the interpretability of machine learning models for transformer diagnostics. Existing studies show that integrating domain knowledge with data-driven models can improve both diagnostic reliability and practical applicability \parencite{Nguyen2025, Souvannalath2024}. 

Finally, this chapter identifies research gaps in current approaches, particularly in the integration of accurate classification models with interpretable and physically meaningful diagnostic frameworks. These gaps form the basis for the proposed methodology presented in the subsequent chapter.

\section{Overview of Transformer Fault Diagnosis Using DGA}
Machine learning (ML) has been widely applied in transformer fault diagnosis using Dissolved Gas Analysis (DGA), where gas concentration data are used to classify fault types. Ren et al. proposed a CatBoost-based fault classification model optimized using SCGWO, demonstrating improved classification performance across multiple transformer fault categories \parencite{Ren2025}. Similarly, Kumar et al. conducted a comparative study of multiple machine learning models for DGA-based transformer fault classification, confirming the effectiveness of classification approaches in identifying fault types \parencite{Kumar2024}. Nguyen et al. further applied adaptive fuzzy logic for DGA-based diagnosis, providing interpretable rule-based classification aligned with transformer fault conditions \parencite{Nguyen2025}. These studies confirm that ML-based classification is a core approach in transformer fault diagnosis.

However, a key limitation identified across these studies is the presence of imbalanced datasets. Du et al. addressed this issue by proposing a reduced-dimensional generative adversarial network (GAN) to generate synthetic minority-class samples, improving class distribution in DGA datasets \parencite{Du2024}. Kari et al. further developed an improved auxiliary classifier GAN (ACGAN) model to enhance data balance and improve diagnostic performance under imbalanced conditions \parencite{Kari2023}. In addition, Chen et al. proposed a sequential ensembled extreme learning machine (SE-ELM) specifically designed for highly imbalanced multiclass DGA classification, demonstrating improved classification stability across both majority and minority fault classes \parencite{Chen2023}. These findings indicate that handling class imbalance is essential for reliable ML-based transformer fault classification.

Another limitation in DGA-based ML models is the availability of labeled data. Zhou et al. proposed a semi-supervised graph convolutional network (GCN) to utilize both labeled and unlabeled DGA data, improving classification performance under limited labeled conditions \parencite{Zhou2023}. Similarly, Sun et al. introduced a dual-stage semi-supervised learning framework to enhance diagnostic performance when labeled fault data are insufficient \parencite{Sun2025}. Liu et al. also applied self-organizing neural networks to model DGA data structures without relying entirely on labeled samples \parencite{Liu2025}. These studies demonstrate that transformer fault diagnosis is not only a classification problem but also a data-constrained learning problem.

Feature representation has also been identified as a critical factor affecting model performance. Peng et al. proposed a multi-dimensional gas representation approach to improve early fault warning capability by capturing relationships among multiple gas features \parencite{Peng2024}. Lin et al. applied manifold-based feature selection to identify relevant gas features and reduce redundancy in transformer fault diagnosis \parencite{Lin2026}. Ren et al. further demonstrated that optimized feature inputs improve the performance of CatBoost-based models for DGA classification \parencite{Ren2025}. These studies show that effective feature representation and selection directly influence classification accuracy.

More advanced approaches have explored data-driven feature learning techniques. Vidal and Castro proposed a hybrid model combining variational autoencoders (VAE) and neural networks to learn compact feature representations from DGA data \parencite{Vidal2023}. Zhou et al. applied graph-based learning to capture relationships between samples in semi-supervised settings \parencite{Zhou2023}. Sun et al. further demonstrated that semi-supervised frameworks can extract useful representations from both labeled and unlabeled datasets \parencite{Sun2025}. These methods indicate a shift from predefined gas features to learned representations in transformer fault diagnosis.

Despite improvements in classification accuracy, interpretability remains a challenge. Wang et al. highlighted that hybrid AI models for DGA-based diagnosis may lack transparency in their decision-making process \parencite{Wang2023}. Nguyen et al. demonstrated that fuzzy logic models provide interpretable diagnostic rules based on gas features \parencite{Nguyen2025}. Souvannalath et al. proposed a cognitive artificial intelligence framework that integrates domain knowledge with data-driven learning to improve diagnostic reasoning \parencite{Souvannalath2024}. These approaches indicate that interpretability in transformer fault diagnosis should align with known gas generation mechanisms and fault behavior.

Overall, the reviewed studies show that ML has significantly improved transformer fault classification using DGA. However, the literature consistently identifies key challenges, including class imbalance, limited labeled data, feature representation, and model interpretability. Addressing these challenges is essential for developing reliable and practical transformer diagnostic systems.

\section{Explainable Artificial Intelligence (XAI) and Physical Interpretability}
Explainable Artificial Intelligence (XAI) is applied in ML-based transformer fault diagnosis to improve the interpretability of model predictions. In DGA-based applications, interpretability refers to the ability of a model to provide explanations that are consistent with known gas generation mechanisms associated with transformer faults. Wang et al. showed that hybrid AI models may lack transparency in their decision-making process, which can limit practical adoption \parencite{Wang2023}. Nguyen et al. applied adaptive fuzzy logic systems to produce interpretable rule-based outputs based on gas features \parencite{Nguyen2025}. Souvannalath et al. further proposed a cognitive artificial intelligence framework that integrates domain knowledge with data-driven learning to improve diagnostic reasoning \parencite{Souvannalath2024}. These studies indicate that interpretability is necessary for reliable application of ML models in transformer diagnostics.

Several studies have proposed approaches that integrate interpretability with model performance. Souvannalath et al. introduced a cognitive AI approach that combines contextual reasoning with machine learning models \parencite{Souvannalath2024}. Vidal and Castro developed a hybrid model combining variational autoencoders and genetic programming to improve feature representation and model interpretability \parencite{Vidal2023}. Sun et al. demonstrated that semi-supervised learning frameworks can improve both classification performance and the understanding of data structure in DGA datasets \parencite{Sun2025}. These approaches show that interpretability is incorporated into model design rather than applied as a separate post-processing step.

In DGA-based fault diagnosis, interpretability must also be consistent with physical fault mechanisms. Gas generation patterns are associated with specific fault types, and diagnostic models should reflect these relationships. For example, acetylene is associated with arcing faults, while methane and ethylene are associated with thermal faults. Ensuring that model outputs align with these relationships improves the reliability of diagnostic decisions. Wang et al. emphasized that model transparency is necessary for engineering applications \parencite{Wang2023}, while Nguyen et al. demonstrated that rule-based systems can reflect domain knowledge in diagnostic outputs \parencite{Nguyen2025}. These findings indicate that XAI in DGA-based transformer diagnosis should ensure consistency between model predictions and known physical behavior.

\section{Research Gap}
Based on the reviewed literature, several limitations remain in machine learning-based transformer fault diagnosis using Dissolved Gas Analysis (DGA).

Firstly, although many studies have improved classification accuracy using advanced machine learning models such as CatBoost, GAN-based augmentation, and semi-supervised learning, most approaches focus primarily on performance without sufficiently addressing model interpretability. While methods such as fuzzy logic and cognitive artificial intelligence provide interpretable outputs, they are often not integrated with high-performance models, resulting in a trade-off between accuracy and interpretability \parencite{Ren2025, Nguyen2025, Souvannalath2024}.

Secondly, class imbalance remains a major issue in DGA datasets. Although techniques such as GAN and ACGAN have been proposed to generate synthetic data, these methods increase model complexity and may introduce uncertainty in generated samples. In addition, imbalance-aware models such as SE-ELM improve classification stability, but a unified approach that balances performance and efficiency is still lacking \parencite{Du2024, Kari2023, Chen2023}.

Thirdly, many models are developed and evaluated on specific datasets, which limits their generalization capability. Studies have shown that model performance varies depending on dataset characteristics and operating conditions, indicating that robust validation and generalization strategies are still required \parencite{Wang2023, Zhou2024, Laburu2024}.

Finally, although recent studies have explored explainable and cognitive AI approaches, there is still limited integration between machine learning models and physical knowledge of transformer fault mechanisms. Most models focus on prediction accuracy without ensuring that their outputs are consistent with known gas generation behavior.

Therefore, there is a need to develop a machine learning-based DGA fault classification framework that not only achieves high classification accuracy but also ensures interpretability, handles imbalanced data effectively, and aligns model predictions with physical fault mechanisms.

\section{Chapter Summary}
This chapter reviewed existing studies on machine learning-based transformer fault diagnosis using Dissolved Gas Analysis (DGA). The literature shows that machine learning techniques, including ensemble models, semi-supervised learning, and data augmentation methods, have improved fault classification performance by modeling complex relationships in gas data.

The review also highlighted key components of ML-based diagnostic systems, including problem formulation, feature representation, dataset characteristics, and model development. In particular, feature engineering and data preprocessing were identified as critical factors influencing model performance. Additionally, challenges such as class imbalance, limited labeled data, and model generalization were discussed.

Furthermore, the importance of interpretability was emphasized, where explainable artificial intelligence (XAI) approaches aim to ensure that model predictions are consistent with known transformer fault mechanisms. Despite these advancements, several limitations remain, including the lack of integration between high-performance models and interpretable frameworks.

Based on these findings, a research gap was identified, highlighting the need for a unified approach that combines accurate classification, effective data handling, and physically meaningful interpretation. This forms the foundation for the proposed methodology in the next chapter.
